# SBTI-
网上很火的sbti测试复刻版，仅供学习参考

SBTI 人格测试
MBTI已经过时，SBTI来了。

简介
一个纯前端的中文人格测试页面，包含 31 道精心设计的题目 和 15 维度评分系统，最终生成 12 种独特的人格类型。

预览
SBTI 人格测试 SBTI 人格测试 SBTI 人格测试

快速开始
无需安装任何依赖，直接在浏览器中打开 index.html 即可。

也可以使用本地服务器启动：

# Python
python -m http.server 8080

# Node.js
npx serve .
然后访问 http://localhost:8080
