# SBTI 人格测试

> MBTI已经过时，SBTI来了。

一个有趣的中文人格测试，包含31道精心设计的题目，生成12种独特的人格类型。

## 在线体验

直接在浏览器中打开 `index.html` 即可开始测试。

## 人格类型

| 代码 | 名称 | 描述 |
|------|------|------|
| CTRL | 拿捏者 | 关系中的主导者，擅长把握节奏和方向 |
| SHIT | 愤世者 | 对世界有清醒认知，看透虚伪和套路 |
| JOKER | 小丑 | 生活中的喜剧演员，善于用自嘲化解尴尬 |
| FISH | 咸鱼 | 人生信条是"躺平即正义" |
| LOVE | 恋爱脑 | 不折不扣的浪漫主义者 |
| COOL | 高冷 | 给人疏离和神秘的印象 |
| SIMP | 舔狗 | 有一颗真诚热烈的心 |
| GIGA | 大屌丝 | 屌丝文化的集大成者 |
| NORM | 正常人 | 最"正常"的人，保持相对平衡 |
| MYST | 谜语人 | 让人捉摸不透的存在 |
| EDGE | 边缘人 | 习惯在人群边缘游走 |
| BOSS | 扛把子 | 天生的领导者 |

## 特色

- 31道精心设计的中文题目
- 15维度人格评分系统
- 纯前端实现，无需后端
- 响应式设计，支持移动端
